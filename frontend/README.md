# One Health — Frontend

Next.js marketing site and admin UI for the One Health Student Society.

**Full setup instructions (Docker, backend, env vars, auth): see the [root README](../README.md).**

## Quick commands

```bash
npm install
cp .env.example .env.local   # Windows: Copy-Item .env.example .env.local
npm run dev                  # http://localhost:3000
```

Requires the NestJS backend running on port `3001` and Docker services started from the project root.
